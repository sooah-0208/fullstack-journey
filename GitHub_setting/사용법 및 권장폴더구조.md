## VSCode Gemini 프롬포트
**Step 1. 가이드 파일 학습시키기**
"내 프로젝트 루트에 있는 AI_GUIDE.md 파일을 읽어줘. 앞으로 내가 요청하는 모든 코드는 이 가이드의 기술 스택, 폴더 구조, 코딩 컨벤션을 엄격하게 준수해야 해. 이해했으면 핵심 규칙을 요약해줘."

**Step 2. 기능 구현 및 폴더링 요청**
"새로운 '사용자 프로필 수정' 기능을 만들 거야. AI_GUIDE.md 구조에 맞춰서 필요한 FE/BE 파일 목록을 먼저 알려주고, app/api/v1/endpoints/users.py에 들어갈 백엔드 로직부터 코드를 작성해줘."

**Step 3. 자동 PR 및 커밋 메시지 생성**
"방금 수정한 코드들에 대해 GitHub PR을 올리려고 해. 변경 사항을 분석해서 feat: 사용자 프로필 수정 기능 추가와 같은 형식의 커밋 메시지와, 주요 변경 내용을 요약한 PR 본문을 마크다운으로 작성해줘.
