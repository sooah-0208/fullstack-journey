현재 열려 있는 FastAPI 엔드포인트 코드를 분석해서, 프론트엔드에서 즉시 사용할 수 있는 API 세트를 만들어줘.

TypeScript Interface: 요청(Request)과 응답(Response) 데이터 타입을 정의해줘.

API Function: Axios를 사용하는 비동기 함수를 작성해줘.

React Query Hook: useQuery 또는 useMutation을 사용하는 커텀 훅 형태로 만들어줘.

출력 형식은 우리 프로젝트 가이드(#AI_GUIDE.md)의 폴더 구조와 컨벤션을 준수하고, 코드 블록 기호는 ```로 표시해줘.

API 개수가 5개가 넘으면 **전체 엔드포인트를 요약해서 테이블 형태로 정리해줘**
