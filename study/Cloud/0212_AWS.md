# AWS란
amazon web services  

콘솔홈-> EC2(얘가 컴퓨터라고 생각하면 됨)-> 인스턴스 시작(새 PC 생성) -> OS 선택
이렇게 가상 서비스 가능

## 과정
1. OS선택
원하는 os로 시작

2. 인스턴스 유형
프리티어로 시작

3. 키 페어
aws가 내 pc에 접속할 때 들어올 키(생성 후 어디 저장해두면 됨)

4. 네트워크 설정
SSH 트래픽 허용 => 터미널에서만 접속 허용
HTTPS 허용 => 브라우저에서 접속 허용

5. 스토리지
이후에 증가 시킬 수 있으니 작게 시작해도 됨

6. 고급 세부정보
도메인 조인 가능(내가 산 도메인으로 시작할 수 있음)
