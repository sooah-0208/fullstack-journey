# 정규표현식
- 특정 규칙이 있는 문자열에서 원하는 문자만 찾기위한 식
    - 전화번호, 이메일, 한글, 오타수정 등 많은 문자열 중에 원하는 값만 추출하기 위함
- python 내장 lib
```
import re

# 조건에 맞는 첫번째 문자열 찾기
re.search([조건], 전체문자열)
print(result.group()) # 찾은 글자
print(result.start()) # 시작 위치(글자 시작 인덱스)
print(result.end())   # 끝 위치(글자 끝 인덱스(공백포함))

# 조건에 맞는 모든 문자열 찾기
re.findall([조건], 전체문자열)
# 조건에 맞는 문자열 제거하기
re.sub([조건], 전체문자열)
```
- 이메일 찾기
```
re.search('[a-zA-Z0-9._%+-]+@[a-zA-Z0-9]+\.[a-z]+', text)
```
- 전화번호 찾기
```
re.findall(r'\d{2,3}-\d{3,4}-\d{4}', text)
```
- `\d`: 디지털 숫자(0-9) = `[0-9]`
- `{숫자}`: `\d`의 개수, `,`로 범위지정가능
- `-`: 문자 그대로 하이픈 포함한 거 찾아줌


## 기본 문법
| 기호 | 의미 | 예시 | 설명 |
| :---: | :--- | :--- | :--- |
| **`.`** | 임의의 한 글자 | `a.b` | `aab`, `abb`, `acb` 등 a와 b 사이에 아무 글자가 하나 있는 패턴 |
| **`.`** | 임의의 한 글자 | `a.b` | `aab`, `abb`, `acb` 등 a와 b 사이에 아무 글자가 하나 있는 패턴 |
| **`.*`** | 사이의 여러글자 | `a.*b` | `afefb`, `abb`, `adsfgb` 등 a와 b 사이에 아무 글자가 여러개 있는 패턴 |
| **`*`** | 0번 이상 반복 | `a*b` | `b`, `ab`, `aab`, `aaab` 등 a가 없거나 여러 번 반복되고 b가 오는 패턴 |
| **`^`** | 문장의 시작 | `^Hello` | "Hello"로 시작하는 문장만 찾음 |
| **`$`** | 문장의 끝 | `Done$` | "Done"으로 끝나는 문장만 찾음 |
| **`[ ]`** | 문자 선택 범위 | `[0-9]` | 0부터 9 사이의 숫자 중 무작위 하나를 찾음 |
