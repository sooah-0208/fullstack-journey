# vsCode로 Docker 접속하는 방법

## 데이터 파일 다운로드

compose.yml파일에 각 파일 volumes의 파일 경로 확인 후 각 폴더 위치 맞춰주기

DDL.spl 파일에 사용자 정보 추가 및 수정
```
use mysql;

CREATE USER 'kimjihwan'@'%' IDENTIFIED BY 'jihwan'; 
GRANT ALL ON edu.* TO 'kimjihwan'@'%';
FLUSH PRIVILEGES;
```
`CREATE USER`: DB사용자 계정 생성 명령
'원하는 아이디'
`@`: 연결부, @앞의 아이디와 뒤의'%'조건을 연결시킴. 조건이 맞아야만 해당 id로 접속할 수 있게 만듦
`'%'`: 모든 IP, 모든 Host(로컬PC, 다른PC, 서버, 도커 컨테이너 밖) 어디서든 접속 가능하게 만드는 조건문
-> `'localhost'`로 바꾸면 다른 PC에선 접속 못 하고 로컬호스트에서만 접속 가능해짐
`IDENTIFIED BY`: 비밀번호 설정



## 데이터 폴더의 파일들 삭제

정보 추가 docker compose up -d
정보 삭제 docker compose down
