java method
```
 public static void main(String[] args) {
    
 }
```
여기서부터가 이 프로젝트의 시작이라고 알려줌, 이게 있어야 돌릴수있음
public: 접근제한자, public은외부에서도 접속 가능
static: 메모리에 올라가는 대상(시작점)
void: return type중 하나, return하지 않겠다는 의미
main~: 우리가 흔히 아는 함수
이 중괄호 안에 실행시킬 함수를 넣으면 됨
