# 🌐 JavaScript로 웹 화면 조작하기

---

## 1️⃣ JS 파일 연결하기

### 📄 JS 파일 생성 후 HTML에 연결
```html
<head>
  <script src="script.js"></script>
</head>
```

❓ 어? 안 나오는데요?

### ❗ 이유
- 코드는 **위에서부터 순서대로 실행**
- script가 태그보다 위에 있으면
- 아직 태그가 없는 상태에서 실행 → 오류 발생

### ✅ 해결 방법
```html
<body onload="함수이름()">
```

### 📌 onload란?
- body 안의 **모든 태그가 실행된 후**
- `onload="얘"` ← 이 함수 실행하겠습니다 라는 의미

---

## 2️⃣ 콘솔에서 태그 위치 찾는 법

```js
document.getElementsByTagName('section')[0];
```

### ❓ 왜 `[0]`이 붙을까?
- body 안에 같은 태그가 **여러 개 있을 수 있음**
- 그래서 기본적으로 **배열(Array) 자료형**
- 그중 첫 번째 요소를 가져오기 위해 `[0]` 사용

---

## 3️⃣ document에 요소 넣는 방법

```js
document.getElementsByTagName("section")[0].innerHTML = 요소명;
```

- `innerHTML` : 태그 안에 내용을 넣거나 바꾸는 속성
