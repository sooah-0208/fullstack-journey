# 1️⃣ `<nav>` + `<a>`
메뉴 화면에 사용하는 코드 => 이동 메뉴 영역이라고 알려주는 태그
해당 위치로 이동시켜준다

## ✅ <a> (링크) — 이동용
```
<nav>
  <a href="#home">Home</a>
</nav>
```
- 페이지 이동 / 스크롤 이동
- 새 페이지 열기 가능
- 내비게이션 메뉴에는 정석
- SEO, 접근성 👍 (SEO란? 검색엔진 최적화: 내 사이트 구조를 검색엔진이 이해하기 쉽게 만드는 것)
- 👉 메뉴라면 기본은 a

# 2️⃣ <a href="#home">
### ✅ **“id가 home인 위치로 이동하라”**
즉 id="home"을 만들어줘야함(보통 section코드에 넣어줌, div도 가능)

# ✅ “진짜 홈 화면”으로 가고 싶다면?
1. 새 페이지
`<a href="index.html">Home</a>`
2. 루트로 이동
`<a href="/">Home</a>`
3. 현재 페이지 맨 위
`<a href="#">Home</a>`
