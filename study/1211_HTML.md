# HTML
### 1) 태그 < >
 - `H1 /H1`, `H2 /H2`...: 폰트 크기(제목 소제목 느낌)
  
 - `div`: 영역 정의
  
  <img width="219" height="258" alt="image" src="https://github.com/user-attachments/assets/228e3072-2f50-4c02-802a-63407732b352" />
  
 - `p /`: 글자영역
 - `li /`: 리스트
 - `ol /`: 순서가 있는 목록(자동으로 숫자 입력)
 - `nav/`: 링크 모음(웹패이지 주요 링크 모음을 나타낼 때 사용)
 - `a /`: 링크를 만드는 태그 -> 속성: href(하이퍼링크)
   
 ```
<nav>
  <a href="index.html">홈</a>
  <a href="about.html">소개</a>
  <a href="contact.html">문의</a>
</nav>
```
