## 배열`[요소,요소,...]`
- array `arr`
- 배열은 무조건 순차적 => index(위치값)을 가지고 있음
- 쉼표`,`로 요소를 구분함
- `[]`형태면 배열인가? 의심해야함
- 자료형을 구분하지만 한 배열 안에 다양한 자료형 값을 가질 수 있음

- 변수[인덱스] => 해당 인덱스 번호의 값 출력  
  변수만 => 전체 배열 출력
- 반대로 `변수[인덱스]='요소'`로 값을 입력할 수도 있음

### 🤔 `const`로 설정하면 값을 못 바꾸나요?
구조는 변경할 수 없지만 안의 요소는 변경 가능하다!   
그래서 ⭐배열(객체)을 사용할 땐 const가 더 안전함⭐
