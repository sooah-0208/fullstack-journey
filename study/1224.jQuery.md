jQuery 안에는
function name(){} / ()=>{}(에로우함수) 두가지 사용 가능함


$('section').html(html);
-> $(대상): 'section'이라는 놈에게 
-> .(함수): html 형식으로
->(요소): 요소를 넣어줘

이벤트는 중복이 가능함
=> 중복을 없애기 위해
on off 가 필요함
$(태그/클래스/#아이디).off() -> 모든 이벤트를 없앤다
$(태그/클래스/#아이디).off().on("실행된 이벤트") -> ""안의 이벤트를 없앤다
