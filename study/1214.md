### a href
<a href=""] 클릭 </a] 하세요
여기서 `<a>`는 `클릭`을 누르면 이동시키게 만드는 장치

<a href="https://www.naver.com/" target"=_blank" title="네이버로 이동"]
`target"=_blank"`는 새 창으로 띄우겠다는 의미
`title="네이버로 이동"`로 title을 정해주는 이유는 사용자에게 추가 설명을 주기 위함
->링크에 마우스를 올리면 툴팁(작은 말풍선)으로 ""안의 설명이 뜸

# WS vs WAS
web server(WS)는 정적파일을 서비스하기 위함 프로그램(프론트엔드에 관련된 서버)
web application server(WAS)은 동적파일을 서비스하는 프로그램(백엔드 / 뉴스 광고 등 매번 다르게 화면이 뜨는 것)
