# IF문
```
if(true){

}
```
if문의 반대 => else =>무조건 if가 있어야 사용할 수 있음
```
if(!true){
console.log('참');   
} else{
    console.log('거짓');
}

=>거짓
```
1) **! = 부정**
2) if문의 조건이 하나일 경우 `{}`는 제외하고 사용 가능하다.

# jump = continue
continue를 넣으면 아래 항목을 건너띔  
=> 생략하고 싶을 때 사용
```
for(let i =0 ; i<10; i++) {
    if(i === 0){
        continue;
    }
    if(i%2 == 0){
        console.log(i);
    }
}
```
==> `if(i === 0){continue;}`: i가 0이면 `if(i%2 == 0){continue;}
        console.log(i);`를 뛰어넘는다.(=생략한다.)

## 삼항연산자
`a = (조건)? 참일 때 표현 : 거짓일 때 표현;`
ex)
```
for(let i =0 ; i<9 ; i++)
{
a= (i % 2 ===0) ? a+'true' : a+'false';
console.log(a)
}

1) a는 true 혹은 false가 됨
2) a의 조건: i를 2로 나눴을 때 나머지가 0인경우
3)  참이면 a+true를 하고 거짓이면(else) a+false를 하라

=> for(let i =0 ; i<9 ; i++)
{ if(i % 2 ===0){
 a+'true'} else{ a+'false'
}
console.log(a);
}
```
