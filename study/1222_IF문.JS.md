# IF문
```
if(true){

}
```
if문의 반대 => else =>무조건 if가 있어야 사용할 수 있음
```
if(!true){
console.log('참');   
} else{
    console.log('거짓');
}

=>거짓
```
1) **! = 부정**
2) if문의 조건이 하나일 경우 `{}`는 제외하고 사용 가능하다.

### 비교자
>, >=, <, <=, %(나머지)


## IF문을 여러개 사용할 때
&&로 연결 = and  
||로 연결 = or  

# jump = continue
continue를 넣으면 아래 항목을 건너띔  
=> 생략하고 싶을 때 사용
```
for(let i =0 ; i<10; i++) {
    if(i === 0){
        continue;
    }
    if(i%2 == 0){
        console.log(i);
    }
}
```
==> `if(i === 0){continue;}`: i가 0이면 `if(i%2 == 0){
        console.log(i);`를 뛰어넘는다.(=생략한다.)
